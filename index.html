  • Model：model（Kepler/TESS、GP 去雜訊、TLS、CNN；含 RF/XGBoost 基準）
  • Frontend：frontend、frontend-test（React/Three.js 視覺化）
  • Backend：back_end_1.0（Flask 推論）
  • Slides：slides；Colab：colab_notebook

文章

  • 應變網路行動車：MEO 衛星 + 相位陣列 — 行動車雙天線 handover、Ka 頻段降頻與基頻處理，轉成 IP 串接地面網路，支援災害備援。
  • SDR Platform：LEO 接收站的軟體化 — 以通用 Down Converter + 軟體基頻處理，整合多頻段訊號 → IP，後端可接 O-RAN / Ethernet / Wi-Fi。
